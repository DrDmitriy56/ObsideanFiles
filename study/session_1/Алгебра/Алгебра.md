77. Система линейных алгебраических уравнений (СЛАУ) — это совокупность из m уравнений, где каждое уравнение является линейным, и все они содержат одни и те же переменные.
78. 
- **Линейное уравнение**: Уравнение вида `a₁x₁ + ... + aₙxₙ = b` или `ax + b = 0`
    
- **Решение линейного уравнения**: Набор значений переменных, обращающий уравнение в верное равенство.
    
- **Система линейных уравнений (СЛАУ)**: Несколько линейных уравнений от одного набора переменных.
    
- **Решение СЛАУ**: Набор значений переменных, удовлетворяющий **всем** уравнениям системы одновременно.
    
- **Совместная система**: СЛАУ, имеющая **хотя бы одно** решение.

79. 
	1. **По количеству решений**:
	
	- **Совместная**:
	
		- **Определённая** (одно решение).
			
		- **Неопределённая** (бесконечно много решений).
		
	- **Несовместная** (нет решений).
	
	1. **По виду**:
	
	- **Однородная** (все свободные члены `b = 0`).
	
	- **Неоднородная**.

80. 
- Две СЛАУ называются **эквивалентными**, если они имеют одно и то же множество решений.
    
- **Основной приём**: Преобразования, не меняющие множество решений (перестановка уравнений, умножение на ненулевое число, сложение уравнений).

81. 
- **Цель**: Привести систему к **ступенчатому (треугольному)** виду с помощью элементарных преобразований строк.
    
- **Алгоритм**:
    
    1. Прямой ход: Исключение переменных ниже главной диагонали.
        
    2. Обратный ход: Нахождение значений переменных, начиная с последнего уравнения.

82. 
- **Матрица** — прямоугольная таблица чисел `(aᵢⱼ)` размера `m × n`.
    
- **Основные операции**:
    
    1. **Сложение/вычитание** (матрицы одинакового размера).
        
    2. **Умножение на скаляр**.
        
    3. **Умножение матриц** `A(m×n) * B(n×k) = C(m×k)` для квадратных
    4. **Единичная матрица** `E` (на главной диагонали 1, остальные 0).

83. 
- **Сложение**: Коммутативность, ассоциативность.
    
- **Умножение на скаляр**: Дистрибутивность.
    
- **Умножение матриц**:
    
    - **Ассоциативность**: `(AB)C = A(BC)`.
        
    - **Дистрибутивность**: `A(B+C) = AB + AC`.
        
    - **НЕкоммутативно**: `AB ≠ BA` (в общем случае).

84. 
- **Определение**: Замена строк на столбцы. Матрица `Aᵀ`, где `Aᵀ[i][j] = A[j][i]`.
    
- **Свойства**:
    
    1. `(Aᵀ)ᵀ = A`
        
    2. `(A + B)ᵀ = Aᵀ + Bᵀ`
        
    3. `(AB)ᵀ = BᵀAᵀ`
        
    4. `(λA)ᵀ = λAᵀ`

86. 
- **Унипотентная матриц** — треугольная матрица с единицами на главной диагонали.
    
- Произведение двух **верхних (нижних)** унипотентных матриц — также **верхняя (нижняя)** унипотентная матрица.

87. 
- **Симметричная**: `Aᵀ = A` (элементы симметричны относительно главной диагонали).
    
- **Кососимметричная (антисимметричная)**: `Aᵀ = -A`. На главной диагонали всегда нули.

88. 
- Любую квадратную матрицу `A` можно **единственным образом** представить в виде суммы:  
    `A = S + K`, где  
    `S = (A + Aᵀ)/2` — симметричная часть,  
    `K = (A - Aᵀ)/2` — кососимметричная часть.
    

89. 
- **Определение**: Матрица, разбитая на более мелкие прямоугольные блоки (подматрицы).
    
- **Умножение блочных матриц**: Выполняется аналогично обычному, если размеры блоков согласованы (как для обычных матричных элементов).

90. 
- **Определитель** — это числовая характеристика квадратной матрицы `(det(A))`, задаваемая как единственная функция, удовлетворяющая трём аксиомам:
    
    1. **Мультилинейность** по каждой строке (столбцу).
        
    2. **Антисимметричность** (перестановка двух строк меняет знак).
        
    3. **Нормировка**: `det(E) = 1`.

91. 
- `det(λA) = λⁿ * det(A)` (для матрицы `n×n`).
    
- Если есть **нулевая строка/столбец** → `det = 0`.
    
- Если **две строки (столбца) равны** или пропорциональны → `det = 0`.
    
- Определитель **не меняется**, если к строке прибавить другую строку, умноженную на число.
    
- `det(диагональной/треугольной матрицы)` = произведение элементов главной диагонали.
    
- Для **блочно-треугольной** матрицы: `det = det(A) * det(D)` (если матрица вида [ [A, B], [0, D] ]).

92. 
- **Определение**: Для квадратной матрицы `A` матрица `A⁻¹` называется обратной, если:  
    `A * A⁻¹ = A⁻¹ * A = E`.
    
- **Существует** тогда и только тогда, когда `det(A) ≠ 0` (матрица невырожденная).
  
93.  методом гауса
- **Алгоритм**:
    
    1. К исходной квадратной матрице `A` размера `n×n` справа приписать единичную матрицу `E` того же размера. Получится расширенная матрица `[A | E]`.
        
    2. Методом элементарных **преобразований строк** привести левый блок `A` к единичной матрице `E`.
        
    3. В результате этих же преобразований правый блок `E` превратится в искомую обратную матрицу `A⁻¹`.
        
    4. Если левый блок привести к `E` невозможно (появится нулевая строка), то `A` — вырожденная и обратной не имеет.
       
94. 
- **Перестановка** множества `{1,...,n}` — это упорядоченный набор `(σ₁,...,σₙ)` из всех этих чисел.
    
- **Композиция перестановок**: Последовательное выполнение двух подстановок. Результат — новая перестановка.
    
- **Инверсия** в перестановке — пара индексов `(i, j)`, где `i < j`, но `σᵢ > σⱼ`.
    
- **Число инверсий** `N(σ)` — количество всех инверсий в перестановке `σ`.
    
- **Четность перестановки**:
    
    - **Чётная**, если `N(σ)` — чётное число.
        
    - **Нечётная**, если `N(σ)` — нечётное число.
        
- **Знак перестановки (сигнатура)** `sgn(σ)`:
    
    - `sgn(σ) = +1`, если перестановка чётная.
        
    - `sgn(σ) = -1`, если перестановка нечётная.
        
- **Транспозиция** — перестановка, которая меняет местами **ровно два** элемента, а остальные оставляет на месте. Любая транспозиция — **нечётная** перестановка.

95. 
- При транспозиции (**перестановке местами**) двух **соседних** элементов в перестановке число инверсий изменяется на `±1` (а значит, **чётность перестановки меняется**).

==96. ==
- Любая **транспозиция** (далеких элементов) может быть представлена в виде **нечётного числа транспозиций соседних элементов**.
    
- **Следствие**: Любая транспозиция меняет чётность перестановки на противоположную.

97. 
- Определитель квадратной матрицы `A = (aᵢⱼ)` порядка `n` может быть вычислен по формуле:  
    `det(A) = ∑_{σ ∈ Sₙ} [ sgn(σ) * a_{1σ(1)} * a_{2σ(2)} * ... * a_{nσ(n)} ]`
    
- **Пояснение**: Сумма берётся по **всем перестановкам** `σ` множества `{1,...,n}`. Каждое слагаемое — произведение `n` элементов матрицы, взятых по одному из каждой строки и каждого столбца, умноженное на знак перестановки.

98. 
- **Доказательство (ключевая идея)**: Используется **формула Лейбница**.
    
    - Элемент `a_{iσ(i)}` в формуле для `det(A)` соответствует элементу `a_{σ(i)i}` в матрице `Aᵀ`.
        
    - Если рассмотреть перестановку `σ⁻¹` (обратную к `σ`), то `sgn(σ⁻¹) = sgn(σ)`.
        
    - Произведение элементов в слагаемом для `det(Aᵀ)` будет в точности тем же, что и для `det(A)`, но соответствующее другой перестановке. Сумма по всем перестановкам одинакова.
        
- **Вывод**: Определитель не меняется при транспонировании.

99. 
- **Минор** `Mᵢⱼ` элемента `aᵢⱼ` — определитель матрицы, полученной из исходной вычёркиванием `i`-й строки и `j`-го столбца.
    
- **Минор порядка `k`** — определитель квадратной подматрицы `k×k`, полученной вычёркиванием каких-либо `(n-k)` строк и `(n-k)` столбцов.
    
- **Главный минор** — минор, образованный элементами, стоящими на пересечении строк и столбцов с **одинаковыми наборами индексов**. Особый случай: **угловой главный минор** — минор, образованный первыми `k` строками и первыми `k` столбцами.

100. 
**Алгебраическое дополнение** `Aᵢⱼ` элемента `aᵢⱼ` вычисляется по формуле:  
`Aᵢⱼ = (-1)ⁱ⁺ʲ * Mᵢⱼ`,  
где `Mᵢⱼ` — минор элемента `aᵢⱼ`.

101. 
- Определитель матрицы `A` можно вычислить, разложив его по **любой строке** `i` или **любому столбцу** `j`:
    
    - **Разложение по `i`-й строке**: `det(A) = aᵢ₁Aᵢ₁ + aᵢ₂Aᵢ₂ + ... + aᵢₙAᵢₙ`.
        
    - **Разложение по `j`-му столбцу**: `det(A) = a₁ⱼA₁ⱼ + a₂ⱼA₂ⱼ + ... + aₙⱼAₙⱼ`.
        
- **Свойство ортогональности**: Сумма произведений элементов строки на алгебраические дополнения **другой** строки равна нулю: `aᵢ₁Aₖ₁ + ... + aᵢₙAₖₙ = 0` при `i ≠ k`.

102.  
- Пусть в матрице `A` порядка `n` выбран **минор `M` порядка `k`**, расположенный в строках с номерами `i₁<...<iₖ` и столбцах `j₁<...<jₖ`.
    
- **Дополнительный минор к `M`** — это минор `M'` порядка `n-k`, полученный вычёркиванием строк `i₁,...,iₖ` и столбцов `j₁,...,jₖ` (т.е. оставшимися элементами).
    
- **Алгебраическое дополнение к минору `M`**:  
    `A(M) = (-1)^{(i₁+...+iₖ) + (j₁+...+jₖ)} * M'`,  
    где `M'` — дополнительный минор.
    
- **Важно**: Это понятие используется в обобщённой **теореме Лапласа** для вычисления определителя разложением по набору строк или столбцов.
  
103. 
	раложение по строкам
104. 
    - **Формулировка**: Для матриц `A (m×n)` и `B (n×m)` с `m ≤ n` определитель их произведения равен сумме произведений всех соответствующих миноров порядка `m`.
    
- **Частный случай (A и B — квадратные порядка n)**: `det(AB) = det(A) * det(B)`.
    
- **Доказательство (ключевые шаги)**:
    
    1. Рассмотрим блочную матрицу `M = [[A, 0], [-E, B]]`. Её определитель равен `det(A)*det(B)` (определитель блочно-треугольной матрицы).
        
    2. Применим к `M` элементарное преобразование: прибавим ко второму блоку строк первый, умноженный справа на `B`. Получим `M' = [[A, A*B], [-E, 0]]`.
        
    3. Определитель не изменился. Вычислим `det(M')` разложением по последним `n` строкам (в которых много нулей). Получим `det(M') = det(AB) * det(-E) * (-1)^(некоторый знак)`.
        
    4. После упрощения: `det(M') = det(AB)`. Следовательно, `det(AB) = det(A) * det(B)`.
       
105. 
- **Алгоритм**: С помощью элементарных преобразований **строк** (которые меняют знак определителя или умножают его на константу) привести матрицу к **верхнетреугольному виду**.
    
- **Результат**: Определитель равен произведению элементов на главной диагонали, умноженному на `(-1)^k`, где `k` — число выполненных перестановок строк.
  
106. 
- **Идея**: Если элементы определителя зависят от параметра (например, `x`), и можно показать, что определитель как функция обращается в ноль при некоторых значениях параметра (`x=a, x=b,...`), то он делится на `(x-a)(x-b)...`.
    
- **Метод**: Найти эти линейные множители, выделить их, определить оставшийся множитель (часто константу), подставив какое-либо удобное значение.

107. 
- Уравнение: $x_{n+2} = p * x_{n+1} + q * x_n$.
    
- **Шаги**:
    
    1. Составить **характеристическое уравнение**: `λ² = pλ + q`.
        
    2. Найти корни `λ₁, λ₂`.
        
    3. **Общее решение**:
        
        - Если `λ₁ ≠ λ₂` (действительные): $x_n = C₁*λ₁ⁿ + C₂*λ₂ⁿ$.
            
        - Если `λ₁ = λ₂ = λ`: `x_n = (C₁ + C₂*n)*λⁿ`.
            
        - Если комплексные сопряжённые `ρ(cos φ ± i sin φ)`: `x_n = ρⁿ(C₁ cos(nφ) + C₂ sin(nφ))`.
            
    4. Константы `C₁, C₂` находятся из начальных условий.

108.  
- **Идея**: Разложить определитель `D_n` порядка `n` по строке/столбцу так, чтобы получить линейное соотношение, связывающее `D_n` с определителями меньшего порядка (например, $D_n = a*D_{n-1} + b*D_{n-2}$).

109. 
- **Правая обратная** `A_R⁻¹`: `A * A_R⁻¹ = E`.
    
- **Левая обратная** `A_L⁻¹`: `A_L⁻¹ * A = E`.
    
- **Теорема**: Если для **квадратной** матрицы `A` существуют одновременно и правая, и левая обратные, то они совпадают.
    
- **Доказательство**: $A_L⁻¹ = A_L⁻¹ * E = A_L⁻¹ * (A * A_R⁻¹) = (A_L⁻¹ * A) * A_R⁻¹ = E * A_R⁻¹ = A_R⁻¹$

112. 
обратная методом Гаусса
113. 
- Для блочной матрицы `M = [[A, B], [C, D]]` существуют громоздкие формулы, аналогичные формулам для чисел.
    
- **Важный частный случай**: Если `A` обратима, то используется **формула Шура**:  
    `M⁻¹ = [[A⁻¹ + A⁻¹B S⁻¹ C A⁻¹, -A⁻¹B S⁻¹], [-S⁻¹ C A⁻¹, S⁻¹]]`, где `S = D - C A⁻¹ B` (матрица Шура), предполагается, что `S` обратима.

114. 
- Для **невырожденной** квадратной СЛАУ `A * x = b` (где `det(A) ≠ 0`):  
    `xᵢ = det(Aᵢ) / det(A)`, где `Aᵢ` — матрица, полученная из `A` заменой `i`-го столбца на столбец свободных членов `b`.

115. 
- Вектор `v` — **линейная комбинация** векторов `v₁, ..., vₖ`, если найдутся скаляры `α₁, ..., αₖ` такие, что `v = α₁v₁ + ... + αₖvₖ`.

116. 
**Линейная оболочка** `L(v₁,...,vₖ)` — множество **всех** линейных комбинаций данных векторов.

117. 
- Система векторов **линейно зависима (ЛЗ)**, если существует их **нетривиальная** (не все коэффициенты нули) линейная комбинация, равная нулевому вектору.
    
- Иначе система **линейно независима (ЛНЗ)**.

118. 
- Если система содержит нулевой вектор, она ЛЗ.
    
- **Доказательство**: Коэффициент 1 при нулевом векторе и 0 при остальных дают нетривиальную нулевую комбинацию.

119. 
- Любая **подсистема** ЛНЗ системы векторов сама ЛНЗ.
    
- **Доказательство (от противного)**: Если бы подсистема была ЛЗ, то существовала бы её нетривиальная нулевая комбинация. Дополнив её нулевыми коэффициентами для остальных векторов системы, получили бы нетривиальную нулевую комбинацию для всей системы, что противоречит её ЛНЗ.

120. 
- Если каждый вектор системы `A` линейно выражается через векторы системы `B`, и при этом `|A| > |B|` (в `A` векторов больше), то система `A` — **ЛЗ**.

- **Задача**: Для заданных узлов `x₀, ..., xₙ` (попарно различных) и значений `y₀, ..., yₙ` найти полином `P(x)` степени ≤ `n` такой, что `P(xᵢ) = yᵢ`.

- **Обоснование**: Сводится к СЛАУ относительно коэффициентов полинома. Её матрица — **матрица Вандермонда**, определитель которой отличен от нуля при различных `xᵢ`. Следовательно, решение существует и единственно.
