мы знаем что n $\perp$ b как вектор главной норали и бинормаль => nb = 0
вспомним что 
	t = r(s)'
	t' = r(s)'' - вектор главной нормали при естественной параметризации => n, n $\perp$ b => nb = 0
-
и тогда 0 = 0 + v,
	v = 0

тогда (7) заменяем v = 0:

n - единичный вектор поэтому
